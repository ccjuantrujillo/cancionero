# Configuración del proyecto 

## Requisitos
```
- Php 7.3.17 
- MariaDB 10.3.39 
- Laravel 7.30 
- Nodejs version 14 
```

## Creamos un proyecto en laravel 7
```
composer create-project laravel/laravel:^7.0 example-cancionero 
cd example-cancionero
php artisan --version (Verifica la versión de laravel)

```
## Ultimo paso
```
npm install
npm run dev

```
